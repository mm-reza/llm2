
# Question and Answer System Based on Google Palm LLM and Langchain

This is an end to end LLM project based on Google Palm and Langchain. This system will provide a streamlit based user interface to ask questions and get answers. 

## Project Highlights

- Use a real CSV file of FAQs
- LLM based question and answer system that can reduce the workload of human.
- This system answers within seconds

## Libraries
  - LLM : Google Palm
  - Framework : LangChain
  - User Interface : Streamlit
  - Text embedding : Huggingface instructor embeddings
  - Vector databse : FAISS
